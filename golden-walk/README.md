# Gomi's Golden Walk

A 45-second web game for Gomi's Golden Days, published at `/golden-walk/`.

## Play

Open `index.html` in a modern browser. Tap **Take a Walk**, then tap the golden moments that appear as little Gomi walks.

For a six-second QA round, open `index.html?test=1` through a local web server.

## Game scope

- Responsive portrait-first layout
- 45-second gentle collection game
- No failure state or penalty
- Baby Gomi character based on the supplied day-three photographs
- Real day-three photograph revealed at the end
- Replay and Web Share support
- Links back to the Gomi's Golden Days home, calculator, and senior care guide
